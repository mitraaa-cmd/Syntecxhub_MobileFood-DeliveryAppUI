# Syntecxhub_MobileFood-DeliveryAppUI
# 🍔 FoodieGo - Mobile Food Delivery App UI

A modern, responsive, and visually appealing Food Delivery App UI built using **HTML, CSS, and JavaScript**. This project delivers a premium food ordering experience with interactive components, smooth animations, cart functionality, and a clean user interface inspired by leading food delivery platforms.
